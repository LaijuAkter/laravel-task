# Laravel Reports App

A simple Laravel application for creating and managing reports with user authentication using Breeze.

---

## 🚀 Features

- Laravel 10+
- Authentication via Laravel Breeze
- Reports CRUD (Create, Read, Update, Delete)
- File upload (image)
- Tailwind CSS UI

---

## 🛠️ Requirements

- PHP 8.1+
- Composer
- Node.js and NPM
- MySQL or any supported database
